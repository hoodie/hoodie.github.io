Was mich am meisten an Archlinux nervt ist dass gerade für die x64 Variante viele Pakete nicht vorhanden sind.

Zum Beispiel PDFtk. Damit kann man schnell viele kleine Sachen mit PDFs machen. Das wichtigste Feature ist allerdings, viele PDFs zu einer zusammenfassen zu können (besonders wichtig bei Uni-Skripten).

Um so schöner ist, dass ich jetzt eine kleine leichte alternative dafür gefunden habe, den Stapler, ein kleines cooles Pythonscript.

Wer es sich das ganze über AUR auf Arch installieren will, das PKGBUILD script clonet sich das script direkt von github -> cool…
